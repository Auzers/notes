## 常见名称
1. **Star** :star数量一定程度反应了该项目的受欢迎程度
2. **README.md** :项目基本使用方法的说明
3. **issue** :可对该项目提问题
4. **LICENSE** :该项目证书，使用该仓库前先确认是否满足你的需求是否触犯License
